Allows you to add a default featured image to posts. Just add a default image in your customizer settings !

== Description ==

Visit your blog posts listing page, or a single post page, and open up the WordPress customizer.

Under *Tiny Thumbnail Settings*, choose or upload your default image. That's it !

You can also upload a default thumbnail for pages, if you need to.

Enjoy !

== Frequently Asked Questions ==

= Can I add a default featured image for pages ? =

Yep !
